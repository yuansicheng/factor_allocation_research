# 宏观因子资产配置

主要研究内容：
1. 选择合适的国内大类资产和因子；
2. 使用相同的配置模型分别基于因子和资产进行配置；
3. 对比因子配置和资产配置的回测结果，进行统计学验证；

仓库内文件/文件夹：
- 宏观因子资产配置 论文框架_20221018.docx：研究框架（安老师整理）；
- 资料：相关论文和研报（安老师整理）；
- data：原始数据，包括资产和因子；
- codes：代码
    - test.ipynb：用于结果展示；
    - factor_allocating_strategy.py.bak：之前基于因子与资产统一配置框架复现的代码，完全按照论文公式；
    - 其他：工具类和函数；
